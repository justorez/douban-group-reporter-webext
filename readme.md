# Douban Group Reporter

💢豆瓣小组评论批量举报工具 - 浏览器扩展

## 安装

Edge 扩展商店地址：[上架审核中](#)

## 用法

登录豆瓣，点击扩展进入主界面。如果小组帖子页面进入，自动获取帖子地址并解析，否则请手动填写帖子地址。

筛选用户请输入全名，筛选评论支持模糊搜索，多个关键字需用空格隔开。
